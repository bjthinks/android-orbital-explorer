What are the numbers?

The first number, N, is the electron's energy level or shell number. The second number, l, is the electron's orbital angular momentum, which determines how much the electron is orbiting around the nucleus, versus oscillating radially. Set l to 0 for an "s" orbital, 1 for a "p" orbital, 2, for a "d" orbital, and so on. The third number, m, is the z-component of l and affects how the electron's orbit is aligned.

What is the ℂ or ℝ?

This button selects which generating set or "basis" of orbitals to display. The complex or "ℂ" orbitals each have a well-defined m value, which maximizes the observer's knowledge of the electron's motion. These states are most meaningful for an isolated atom in an external magnetic field, and they are often encountered in the context of quantum mechanics and solutions to Schrödinger's equation. The real or "ℝ" orbitals are combinations of states with two opposite values of m, which has the effect of localizing the electron's position. These states are most meaningful when atoms interact with each other, and they are often discussed in chemistry.

How can I view an s, p, d, or f orbital?

This is determined by the second number, l. Set it to 0 for an s orbital, 1 for p, 2 for d, and 3 for f. If you also set the ℂ/ℝ switch to ℝ, you should get images that match your chemistry reference.

How might real atoms' orbitals differ from what is shown here?

The orbitals shown here are correct for a hydrogen atom (or for an ion like He+ which has a single electron). When there are multiple electrons, their mutual repulsion changes the shapes of the orbitals. These interactions are extremely difficult to calculate. Loosely speaking, the outer parts of orbitals expand in size due to the screening effect of the other electrons, and the orbitals' shapes get distorted so as to keep all the electrons farther away from each other.

This app shows electron states, not atoms. The element is determined by how many protons an atom has in its nucleus, but orbitals have the same shapes regardless of this number. So the simplest answer to this question is, "all of them". This app changes the number of protons in the nucleus so that all of the orbitals shown are similar in size to each other, which makes it much easier to navigate and explore different states. Specifically, sets the number of protons, Z, equal to the energy level, N. This simplifies internal calculations, and gives all the orbitals the same energy (13.6 eV) and similar radial sizes.

Why do the colors change?

Color cycling is a way of visually representing the time dependence of the electron's wave function, which is a rotation in the complex plane. The rotation speed depends on the electron's energy. Since all of the orbitals shown have the same energy, they all cycle colors at the same rate. To make this visible to the eye, the cycling rate has been slowed down by a factor of 30 quadrillion from what it would be in an actual atom. (For comparison, the size of the atom has been enlarged by "only" a factor of about one billion).

How does the rendering work?

The fragment shader approximates an integral along the line of sight using weighted Gaussian quadrature. The quadrature nodes and weights, as well as the values of the wave function, are precomputed and stored in textures. These textures are all small enough to fit in the fastest level of GPU cache. Even so, numeric integration in a fragment shader is expensive, so this computation is done on a smaller scale (2dp centers), then interpolated, enlarged to screen size, and colored using the CIELUV 1976 perceptual color model.
